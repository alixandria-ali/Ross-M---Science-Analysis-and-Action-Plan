## Study Tips

### ✅ Effective Strategies

- **Active Learning:** Take notes, explain concepts out loud
- **Complete Everything:** Don't skip videos or practice questions
- **Review All Answers:** Learn from both correct and incorrect responses
- **Use Flashcards:** Create cards from Quick Reference sections
- **Study Consistently:** Daily 60-90 min is better than weekend cramming
- **Follow Priority Order:** Start with high priority topics first

### ⚠️ Common Mistakes to Avoid

- Passive reading without practicing
- Skipping video tutorials
- Only reviewing questions you got wrong
- Studying topics in random order
- Cramming the night before
- Not tracking your progress
